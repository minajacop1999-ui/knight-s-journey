# اسم لعبتي 🎮

## المشاهد الموجودة
- **MainMenu.tscn**: القائمة الرئيسية مع أزرار (Start, Options, Quit)
  

## أكواد الأزرار
- `MenuButtons.gd`: التحكم في أزرار القائمة
- `StartButton.gd`: منطق بدء اللعبة

## كيفية التشغيل
1. حمل المشروع
2. افتح Godot Engine
3. استورد المشروع (Import)
4. شغل المشهد MainMenu.tscn

## متطلبات التشغيل
- Godot 4.6
